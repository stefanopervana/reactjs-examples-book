Summary

En este capítulo, usted aprendió sobre diversas herramientas de las herramientas React ecosistema de desarrollo, prueba y producción que hemos utilizado en diversas etapas de nuestro desarrollo de aplicaciones. Hablamos de Babel, la transpiler JavaScript, para convertir nuestra próxima generación de código JavaScript para ES5. También vimos cómo utilizar ESLint y React herramientas dev que hacen que el desarrollo React fácil. Al final, hemos visto cómo webpack se podes utilizar con React con sus potentes cargadoras y opciones de configuración. Vimos cómo estas herramientas podes hacer que el desarrollo de aplicaciones React una hermosa experiencia.

In this chapter, you learned about various tools from the React ecosystem—development, testing, and production tools that we have used in various stages of our app development. We discussed Babel, the JavaScript transpiler, to convert our next generation JavaScript code to ES5. We also saw how to use ESLint and React dev tools that make React development easy. In the end, we saw how Webpack can be used with React with its powerful loaders and configuration options. We saw how these tools can make developing React apps a beautiful experience.
