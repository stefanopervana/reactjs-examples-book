Summary

En este capítulo, nos dimos un vistazo a la refactorización de nuestros componentes. Vimos cómo hacer uso de mixins y extraer funcionalidades similares a utilizar sin problemas en los componentes. También hicimos un vistazo a las interacciones DOM, usando árbitros y relacionados con las acciones de DOM de un componente.

In this chapter, we took a look at refactoring our components. We saw how to make use of mixins and extract similar functionalities to use seamlessly across components. We also took a look at DOM interactions, using refs, and related DOM actions from a component.

En el siguiente capítulo, vamos a explorar cómo React los funciones en el lado del servidor. Veremos cómo React los que nos permite procesar y manejar los componentes en el servidor de HTML, que es útil por varias razones pre-render. También vamos a echar un vistazo a cómo afecta esto a la React ciclo de vida de los componentes.

In the next chapter, we will explore how React functions on the the server side. We will see how React allows us to render and handle components on the server to pre-render HTML, which is useful for several reasons. We will also take a look at how this affects the React component life cycle.
