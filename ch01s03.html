Requiring React library

"Bien entonces! Vamos a empezar. Andá por delante y cambiá el título de la página, por ej., React JS Example. A continuación, tenemos que configurar y requerir la biblioteca React en nuestro archivo."

"Alright then! Let's begin. Go ahead and change the title of the page, to say, React JS Example. Next, we need to set up and we require the React library in our file."

"La página principal de React se encuentra en http://facebook.github.io/react/. Aquí, también vamos a localizar las descargas disponibles para nosotros para que podamos incluirlas en nuestro proyecto. Hay diferentes maneras de incluir y utilizar la biblioteca.

"React's homepage is located at http://facebook.github.io/react/. Here, we'll also locate the downloads available for us so that we can include them in our project. There are different ways to include and use the library.

Podemos hacer uso de bower o instalarlo a través de npm. También podemos incluirlo sólo como una descarga individual, disponible directamente desde el dominio fb.me. Hay versiones de desarrollo que son la versión completa de la biblioteca, así como la versión de producción, que es su versión miniaturizado. También existe la versión del add-on. Vamos a echar, sin embargo, un vistazo a esto más adelante ".

We can make use of bower or install via npm. We can also just include it as an individual download, directly available from the fb.me domain. There are development versions that are full version of the library as well as production version which is its minified version. There is also its version of add-on. We'll take a look at this later though."

"Vamos a empezar con la versión de desarrollo, que es la versión de la fuente unminified React. Añadí lo siguiente a la cabecera del archivo:".

"Let's start by using the development version, which is the unminified version of the React source. Add the following to the file header:"

<script src="http://fb.me/react-0.13.0.js"></script>

"Hecho".

"Done".

"Impresionante, vamos a ver cómo se ve."

"Awesome, let's see how this looks."
