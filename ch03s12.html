Summary

En este capítulo, nos fijamos en la forma de pasar alrededor de los datos en los componentes React con props y estado. También hablamos sobre cómo y cuándo usar el estado y los accesorios. Nos fijamos en cómo propTypes se puede utilizar para validar los props. Después de eso, hablamos del ciclo de vida del componente. Discutimos acerca de los diversos métodos de ciclo de vida y cómo pueden ser utilizados. Después de eso, hemos utilizado estos métodos de ciclo de vida para obtener datos en tiempo real de biblioteca API abierta.

In this chapter, we looked at how to pass around data in the React components using props and state. We also discussed how and when to use state and props. We looked at how propTypes can be used to validate the props. After that, we discussed the component's life cycle. We discussed about the various life cycle methods and how they can be used. After that, we used these life cycle methods to get real-time data from Open Library API.
