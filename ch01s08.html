Summary

En este capítulo, empezamos con React y construimos nuestro primer componente. En el proceso, se estudió la API de nivel superior de React para construir componentes y elementos. Utilizamos JSX para construir los componentes. Vimos cómo mostrar la información estática por medio de React y luego se reemplazó gradualmente toda la información estática con información dinámica usando props. Al final, hemos sido capaces de atar todos los cabos y mostrar los datos simulados en el formato que se devuelve desde la API Cambios Recientes de Abrir biblioteca utilizando React.

In this chapter, we started with React and built our first component. In the process, we studied the top-level API of React to construct components and elements. We used JSX to construct the components. We saw how to display static information using React and then gradually replaced all the static information with dynamic information using props. In the end, we were able to tie all ends together and display mock data in the format that is returned from Open Library's Recent Changes API using React.
