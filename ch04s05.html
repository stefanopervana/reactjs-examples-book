Controlled components

"¿Qué es un componente controlado, Mike?" Shawn se le preguntó.

"What is a controlled component, Mike?" asked Shawn.

"Es un componente de entrada cuyo valor es controlado por React. Al proporcionar la prop valor, estamos informando React que el valor de este campo es 'Shawn'. Una vez React ha declarado que es 'Shawn', cualquier entrada del usuario no lo hará tener un efecto como el valor que ya se encuentra en el ". explica Mike.

"It's an input component whose value is controlled by React. By providing the value prop, we are informing React that the value of this field is "Shawn". Once React has declared it to be "Shawn", any user input won't have an effect as the value is already set in the ." explained Mike.

"Creo que tenemos que recurrir a la state vez de props?" Pidió a Shawn.

"I think that we have to resort to state instead of props?" Shawn asked.

"Exactamente. A medida que un usuario está interactuando con el campo de entrada, tenemos que actualizar el valor de prop uso state y el onChange controlador de eventos. Podes dar una oportunidad?" sugerido Mike.

"Exactly. As an user is interacting with the input field, we need to update the value prop using state and onChange event handler. Can you give it a try?" suggested Mike.

// src/index.js

var InputExample = React.createClass({
  getInitialState() {
    return (
      { name: '-'}
    );
  },

  handleChange(event) {
    this.setState({ name: event.target.value });
  },

  render() {
    return (
      <input type="text"
             value={this.state.name}
             onChange={this.handleChange} />
    );
  }
});

"Impresionante. Este patrón de hacer que el valor de una entrada basada en el estado y actualizar el estado en base a la interacción del usuario hace que sea muy fácil de responder a las interacciones del usuario. También podemos hacer algunas cosas de mantenimiento, tales como la validación de la entrada." Mike explica con más detalle.

"Awesome. This pattern of making the value of an input based on state and updating the state based on the user interaction makes it very easy to respond to user interactions. We can also do some housekeeping things such as validating the input." Mike explained further.

"Por ejemplo, podes cambiar el caso de todo el texto en mayúsculas." Mike añadió.
