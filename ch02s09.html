"Ah, bien. Vamos a hacer eso. React nos permite pasar estilos a los componentes de la misma manera que podes pasar los Props. Por ejemplo, queremos que nuestros títulos sean del color floral white y tal vez queremos cambiar el tamaño de la fuente. Vamos a representarlo de una típica manera CSS de la siguiente manera:"

"Ah, right. Let's do that. React allows us to pass styles to the components the same way props can be passed. For example, we want our headings to be of the floral white color and maybe we want to change the font size. We will represent it in a typical CSS way as follows:"

background-color: 'FloralWhite',
font-size: '19px';

"Podemos representar esto como un objeto JavaScript en la forma CamelCase".

"We can represent this as a JavaScript object in the CamelCase fashion."

    var headingStyle = { backgroundColor: 'FloralWhite',


                         fontSize: '19px'

 
                       };

Entonces, podemos usarlo en cada componente Heading como un objeto JavaScript ".

Then, we can use it in each Heading component as a JavaScript object."

RecentChangesTable.Heading = React.createClass({


  render: function() {


    var headingStyle = { backgroundColor: 'FloralWhite',


                         fontSize: '19px' };


    return(<th style={headingStyle}>{this.props.heading}</th>);


  }


});

"Del mismo modo, vamos a cambiar las filas que tienen sus propios estilos."

"Similarly, let's change the rows to have their own styles."

  RecentChangesTable.Row = React.createClass({

  }

"Sí. También las llaves de este objeto de estilo necesita CamelCased, por ejemplo, backgroundColor, backgroundImage, fontSize, y así sucesivamente."

"Yes. Also the keys of this style object need to be CamelCased, for example, backgroundColor, backgroundImage, fontSize, and so on."

"Mike, yo entiendo el estilo en línea, sin embargo, cómo agrego las clases CSS?"

"Mike, I understood the inline styling, however, how to add the CSS classes?"

"Ah justo. Podemos pasar el nombre de la clase como un atributo de las etiquetas del DOM. Vamos a extraer el estilo a una nueva clase CSS recentChangesTable".

"Ah right. We can pass the class name as an attribute to the DOM tags. Let's extract the styling to a new recentChangesTable CSS class."

// css


recentChangesTable {


  background-color: 'FloralWhite',


  font-size: '19px'


}

"Ahora, para aplicar esta clase para nuestro componente, todo lo que tenemos que hacer es pasarlo al componente mediante el expalined className prop".

"Now, to apply this class to our component, all we need to do is pass it to the component using the expalined className prop."

render: function(){

"Como hemos visto anteriormente, React hace uso de los atributos CamelCase. Aquí, el prop className se convertirá en un atributo de clase normal cuando React renderiza el HTML actual."

"As we have seen previously, React makes use of the CamelCase attributes. Here, the className prop will be converted to a normal class attribute when React renders the actual HTML."
