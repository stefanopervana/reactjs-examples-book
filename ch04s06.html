Uncontrolled components

"React también tiene componentes no controlados donde el value prop no se pasa a la entrada".

"React also has uncontrolled components where the value prop is not passed to the input".

render() {
    return (
      <input type="text" />
    );
  }

"En este caso, el valor introducido por el usuario se refleja inmediatamente en la entrada. Para el ajuste de un valor inicial predeterminado, podemos pasar el valor predeterminado value prop, que actuará como el valor inicial para un componente no controlada."

"In this case, the value entered by the user will be immediately reflected in the input. For setting some default initial value, we can pass the default value prop, which will act as the initial value for an uncontrolled component."

render() {
    return (
      <input type="text" defaultValue="Shawn"/>
    );
  }
