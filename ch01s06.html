"Observá cómo estamos utilizando {} aquí. {} se utiliza en JSX para incrustar información dinámica en nuestra plantilla de vista. Podemos utilizarlo para incrustar los objetos JavaScript en nuestros puntos de vista, por ejemplo, el nombre de una persona o título de esta tabla. Como podes ver, lo que estamos haciendo aquí es utilizar la función map de bucle sobre el conjunto de datos que tenemos. Entonces, estamos devolviendo una row (fila) de la tabla, construida a partir de la información disponible en el objeto row (fila) -. Los detalles acerca de cuándo se creó el evento, quien lo creó y la descripción de un evento ".

"Notice how we are using {} here. {} is used in JSX to embed dynamic information in our view template. We can use it to embed the JavaScript objects in our views, for example, the name of a person or heading of this table. As you can see, what we are doing here is using the map function to loop over the dataset that we have. Then, we are returning a table row, constructed from the information available from the row object – the details about when the event was created, who created it and event description."

"Estamos utilizando la sintaxis JSX aquí para construir las filas de la tabla. Sin embargo, no se utiliza como valor de retorno final a partir de la función de render."

"We are using JSX syntax here to construct the rows of table. However, it is not used as the final return value from render function."

"Eso es correcto, Shawn. React con JSX nos permite crear arbitrariamente elementos para ser utilizados en nuestras vistas, en nuestro caso, la creación de formas dinámicas a partir del conjunto de datos que tenemos. Las filas variables contiene ahora una parte de la vista que habíamos usado en un lugar diferente. también podemos construir otro componente de la vista en la parte superior de la misma ".

"That's correct, Shawn. React with JSX allows us to arbitrarily create elements to be used in our views, in our case, creating it dynamically from the dataset that we have. The rows variable now contains a part of view that we had used at a different place. We can also build another component of the view on top of it."

"Esa es la belleza de ello. React nos permite crear de forma dinámica, el uso y la reutilización de las partes de las vistas. Esto es útil para construir nuestros puntos de vista, parte por parte, de una manera sistemática."

"That's the beauty of it. React allows us to dynamically create, use, and reuse the parts of views. This is helpful to build our views, part by part, in a systematic way."
