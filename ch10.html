Chapter 10. Animation

En el capítulo anterior, tomamos un vistazo a React-router y realizó enrutamiento a diferentes niveles. También nos fijamos en el enrutamiento anidada, que pasa alrededor de los parámetros, y cómo Reactá-router mantiene la historia al realizar las tareas de enrutamiento. Hemos aprendido acerca de pasar alrededor de contexto y usando el contexto para hacer React los componentes. Exploramos los modelos de datos y mezclado y les emparejaron con otros marcos de usar como modelos de datos en la red troncal React similar, y se introdujeron a Flux.

In the previous chapter, we took a look at react-router and performed routing at different levels. We also looked at nested routing, passing around parameters, and how react-router maintains history when performing the routing tasks. We learned about passing around context and using context to render React components. We explored data models and mixed and matched them with other frameworks to use as data models in React-like Backbone, and were introduced to Flux.

En este capítulo, vamos a explorar una diversión React complemento, Animación. Vamos a empezar por continuar con nuestra aplicación Pinterest Gato y mejorarlo para apoyar protagonizada y compartir los datos para actualizar los puntos de vista. a continuación, vamos a explorar la adición de controladores para la animación. Veremos cómo los componentes de envolverse para la animación y cómo React añade los manejadores para diferentes eventos. También exploraremos diferentes eventos y cómo podemos mejorar fácilmente nuestra aplicación con el fin de crear efectos impresionantes.

In this chapter, we are going to explore a fun React add-on, Animation. We will start off by continuing with our Cat Pinterest application and enhancing it to support starring and sharing the data to update the views. We will then explore adding handlers for animation. We will see how components get wrapped for animation and how React adds the handlers for different events. We will also explore different events and how we can easily enhance our application in order to create stunning effects.

  • Making changes to the data flow and passing data from react-router links
  • Animación en React
  • Animation in React
  • transiciones CSS
  • CSS transitions
  • grupos de transición
  • Transition groups
  • manipuladores de transición

  • Animating our dynamic component

Al final del capítulo, podremos empezar a animar a nuestros componentes React para diferentes acciones tales como la adición de nuevos contenidos, el cambio de los datos y la posición, y así sucesivamente. También vamos a ser capaces de agregar controladores para diferentes tipos de eventos y explorar diferentes opciones de animación por ahí, aparte de la animación núcleo complemento.

At the end of chapter, we will be able to start animating our React components for different actions such as adding new content, changing data and position, and so on. We will also be able to add handlers for different types of events and explore different animation options out there, other than core animation add-on.
