Un componente pasa por diferentes eventos del ciclo de vida. Ayudan a facilitar cuando debemos inicializar qué parte de un componente o cuándo deben extraerse algunos datos externos.

A component goes through different life cycle events. They help facilitate when we should initialize which part of a component or when should some external data be fetched.
