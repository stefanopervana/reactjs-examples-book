"Bastante práctico. Sin embargo, esto es sólo para facilidad de desarrollo, ¿verdad? ¿Qué pasa cuando hacemos uso de nuestro código?" Shawn preguntó.

"Pretty handy. However, this is just for ease of development, right? What happens when we deploy our code?" asked Shawn.

"JSX no está destinado a ser compilado en tiempo de ejecución. Aunque hay un transformador JSX que convierte JSX a JavaScript en un navegador. Su uso para compilar JSX en tiempo de ejecución en el navegador ralentizaría nuestra aplicación. Vamos a utilizar herramientas tales como Babel, que es un transpiler JavaScript para convertir nuestro código JSX al código JavaScript nativo antes de desplegar la aplicación".

"JSX is not meant to be compiled at runtime. Though there is a JSX transformer that converts JSX to JavaScript in a browser. Using it to compile JSX at runtime in the browser would slow down our application. We will use tools such as Babel, which is a JavaScript transpiler to convert our JSX code to the native JavaScript code before deploying the application."
